J2ME Polish 2.0

2007-02-21: J2ME Polish 2.0 Release Candidate 3 is here. Enjoy these indespensable new features:
  • Overhauled GUI Library: apply effects everywhere: to texts, screen changes, animations, and more - and all this without adding additional resources to your projects! You can also directly access advanced J2ME Polish features while still programming with the usual javax.microedition.lcdui framework. Did we mention the support of subcommands for your menus?
  • Serialization Framework: Want to store or transmit data? Just implement de.enough.polish.io.Serializable and J2ME Polish takes care of the serialization for you.
  • Persistence: Along with the serialization framework we also throw in the RmsStorage that allows you to read and save data in the easiest possible way!
  • Remote Method Invocation: a convenient yet higly efficient method for calling server side functionalities!
  • IDE Integration: With our full blown Eclipse integration and much improved NetBeans support you can concentrate on your work rather than on our tools.
  • Java 5.0 Support: Use Java 5.0 features like enums or generics for your J2ME projects as well!
  • Floating Point Support: Use CLDC 1.1 features like floating point calculations on CLDC 1.0 mobiles as well - J2ME Polish takes care of the necessary conversion.

Please check out the current release candidate.

Overview

J2ME Polish is suite of tools for creating "polished" J2ME applications. Each tool meets a definite need of J2ME developers:
Build-tools with an integrated device-database, a powerful GUI, a framework for building localized applications, a game-engine, a logging framework and a collection of utilities.

These screenshots show the optional GUI of J2ME Polish in action. These pictures show the same application with different designs. Absolutely no changes were made in the source code of the application. Have a look at the screenshots-page for more examples and details.

Status

J2ME Polish is a stable & complete solution for Windows, Linux and Mac OS X. With over 100 downloads per day, J2ME Polish is rapidly becoming the de-facto standard for developing J2ME applications. Please download the latest version (J2ME Polish <%= PolishVersion %>).
