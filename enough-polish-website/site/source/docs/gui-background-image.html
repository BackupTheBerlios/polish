Design

The image background displays an image. 

.myItem {
	background {
		image: url( bg.png );
		color: translucent;
		anchor: top | left;
	}
}

CSS attributes:

CSS Attribute  DefaultValuesExplanation
typen/aimage Set the type to image or specify the background-image attribute.
colorwhitecolor definition The color of the background, can be a rbg, argb or hexadecimal value of the color or the name of the color as defined in the colors section. You can also set the color to translucent, in which case no filling is done.
imagen/aURL The URL of the image.
anchorhorizontal-center| vertical-centerleft, horizontal-center, right, top, vertical-center, bottom You can specify the aligment of the image in case the background is larger or smaller than the image. Mix any of the possible values.
repeatnoneString Either "repeat", "no-repeat", "repeat-x"/"repeat-horizontal" or "repeat-y"/"repeat-vertical". Determines whether the background should be repeated, repeated horizontally or repeated vertically. Default is no repeat.
overlapfalseboolean Defines whether the tiles should overlap over the actual background-area when they don't fit exactly into the actual background-area. Either "true"/"yes" or "no"/"false". Defaults to false. Is only applicable when the repeat mode is activated.
padding-horizontal0integer The horizontal gap between tiles, can be negative for overlapping the tiles. Defaults to 0. Is only applicable when the repeat mode is activated.
padding-vertical0integer The vertical gap between tiles, can be negative for overlapping the tiles. Defaults to 0. Is only applicable when the repeat mode is activated.
