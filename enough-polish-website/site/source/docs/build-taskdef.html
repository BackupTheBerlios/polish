Definition of the J2ME Polish Task

The J2ME Polish Task needs to be defined in the build.xml before it can be used: 

<taskdef 
	name="j2mepolish" 
	classname="de.enough.polish.ant.PolishTask" 
	classpath="import/enough-j2mepolish-build.jar:import/
		jdom.jar:import/proguard.jar:import/retroguard.jar"/>

You should also define the Ant-property "wtk.home" which points to the directory containing the wireless toolkit: 

<property 
	name="wtk.home" 
	value="C:\WTK2.1" />

On a system without the Wireless Toolkit (like Mac OS X) you can still use J2ME Polish by defining the preverify-attribute of the <build>-element instead of using the "wkt.home"-property.
