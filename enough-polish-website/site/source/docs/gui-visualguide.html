This guide shows you the available screens, items and effects that you can use with J2ME Polish. Click on the GUI components to learn more about them.

Screen Elements

There are several typical components in each application's user interface. Hover your mouse over the areas of interest to find out more about them.

1. ScreenInfo
1. Use the ScreenInfo element to display information consistently over all screens of your application.
2. Title
2. There are various options to design screen titles, including text effects.
3. Screen Background
3. Screen backgrounds usually range from the very top to the bottom of the screen, but you can also let them start under the title and stop at the menubar.
4. ScrollBar
4. J2ME Polish uses either a scrollbar (as shown here) or simple scrollindicators to inform the user about possible scrolling.
5. MenuBar
5. Choose between two menubar options that provide different design possibilities.
6. MenuBar
6. With J2ME Polish you can use sub menus as well.
7. Focused Item
7. Use the focused or the [style-name]:hover style selector to design interactive elements when they are focused by the user.
8. Ticker
8. A ticker scrolls information over a screen and can include images in J2ME Polish.

Screens

J2ME Polish caters standard as well as advanced screens:

List

The List screen is used for displaying a menu or for selecting options.

Form

In a form you can embedd any kind of items for building a rich UI.

Alert

Use the alert to notify your user about anything important.

TextBox

The TextBox is used for text input.

TabbedForm

The TabbedForm can contain any kind of items in several tabs.

FramedForm

The FramedForm allows you to use one scrollable main area and several fixed frames.

SnapshotScreen

Popup Screens

Use any screen as a popup screen.

Items

StringItem

Display and design text with StringItem.

TextField

You can enter input using a TextField.

Backgrounds

The simple background displays a monocolored and possibly translucent background.

Round-Rect Background

The round-rect background displays a monocolored background with rounded corners.

Image Background

The image background displays an image in the background.

Vertical Gradient Background

The vertical-gradient background displays an gradient in the background.

Horizontal Stripes Background

The horizontal-stripes background displays horizontal stripes in the background.
