Setting up a J2ME Polish Project

Choose the Mepose Wizard from the New Project menu.

Enter a project name. At the moment it is not possible to convert an existing project so make sure the project is unique. (v0.6.x)

Alter the paths you want to use in this project. At least WTK Home and J2ME Polish Home must be set.

Select the devices you want to support with this project. You can only set them once so make sure you make the right choice. (v0.6.x: This limitation will vanish in future versions)

Alter the Java settings if you like. Note that all possible jars for the selected devices are already included in the project.

Click "Finish" to create the project.

The project is set up!
