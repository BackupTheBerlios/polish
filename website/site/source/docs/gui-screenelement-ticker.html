A ticker scrolls information over a screen and can include images in J2ME Polish.

Design


The above example has been realized with following CSS code: 

colors {
	fontColor: #222;
	fontShadowColor: #fff;
}
.mailTicker {
	ticker-step: 1;
	padding-horizontal: 5;
	layout: vertical-center;
	icon-image: url( sending.png );
	font-color: fontColor;
	font-size: small;
	text-effect: shadow;
	text-shadow-color: fontShadowColor;
}

Specific design attributes:

CSS Attribute  DefaultValuesApplies To  Explanation
ticker-stepinteger2ticker style The number of pixels that the ticker is moved forward in each animation phase.
ticker-positionbottomtop, bottomticker or screen style The position of the ticker - bottom places the ticker above the menubar, top puts the ticker under the title of a screen.

The J2ME Polish ticker also provides all design options of the IconItem.

Programming

You can exchange the image of the ticker by calling UiAccess.setTickerImage( Ticker ticker, Image image ). You can also apply a (new) style to a ticker by calling the screen's setTicker( Ticker ticker ) method with a #style preprocessing directive: 

//#style alarmTicker
form.setTicker( ticker );
