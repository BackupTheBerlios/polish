Mask Background

  • Related Topics:
  • Backgrounds

Cut any background into shape and/or add transparency to it using the mask background.

Design

The above mask background has been realized with following design settings: 

backgrounds {
	maskBackground {
		type: polygon;
		points: 5,90 55,0 105,90 155,0 205,90;
		reference-width: 210;
		reference-height: 100;
		scale-mode: scale;
		color: maskColor;
	}
	maskGradient {
		type: vertical-gradient;
		top-color: white;
		bottom-color: blue;
	}	
}


menubar {
	margin: 0;
	padding-top: 3;
	padding: 2;
	background {
		type: vertical-gradient;
		top-color: blue;
		bottom-color: #fff;
	}
	background {
		type: mask;
		mask: maskBackground;
		mask-color: maskColor;
		background: titleGradient;
		opacity: 150;
	}
}
