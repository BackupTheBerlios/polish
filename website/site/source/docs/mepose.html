Mepose

<%index %>
Mepose is the Eclipse environment for using J2ME Polish. Use syntax highlighting and auto completion for preprocessing, simply debug your MIDlets and setup projects easily with Mepose.

Current Status

Please note that Mepose is currently under heavy development and should not used within production environments right now. If you have any questions or comments, please email us or submit your questions to the discussion board.
There are some known issues/limitations:

  • Debugging: Debugging is currently only really tested with the WTK. Nokia emulators also seem to work.
  • Break-Points: Double clicking in the left frame of the source editor does not work. Please rightclick the left frame to toggle breakpoints.
  • Closing Projects: after opening a closed J2ME Polish project you need to restart Eclipse.
  • Currently you need to convert a J2ME Polish project again for changing the target devices.
  • OS X: Debugging does not work with the Mobile Power Player (yet).
