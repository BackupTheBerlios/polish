Horizontal Stripes Background

  • Related Topics:
  • Backgrounds

Design

Displays horizontal stripes with a gradient. 

.myItem {
	background {
		type: horizontal-stripes;
		first-top-color: green;
		first-bottom-color: green;
		second-top-color: white;
		second-bottom-color: yellow;
	}
}

CSS attributes:

CSS Attribute  DefaultValuesExplanation
typen/a Set the type to horizontal-stripes for having this background.
first-top-colorwhitecolor definition The color of the first stripe at the top of the background, can be a rbg/hexadecimal value or the name of the color as defined in the colors section.
first-bottom-colorblackcolor definition The color of the first stripe at the bottom of the background, can be a rbg/hexadecimal value or the name of the color as defined in the colors section.
second-top-colorwhitecolor definition The color of the second stripe at the top of the background, can be a rbg/hexadecimal value or the name of the color as defined in the colors section.
second-bottom-colorblackcolor definition The color of the second stripe at the bottom of the background, can be a rbg/hexadecimal value or the name of the color as defined in the colors section.
