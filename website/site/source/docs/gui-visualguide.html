Visual Guide

This guide shows you the available screens, items and effects that you can use with J2ME Polish. Click on the GUI components to learn more about them.

GUI Elements

There are several typical components in each application's user interface. Hover your mouse over the areas of interest to find out more about them.

1. ScreenInfo
Use the ScreenInfo element to display information consistently over all screens of your application.
2. Title
There are various options to design screen titles, including text effects.
3. Content
J2ME Polish provides all standard as well as many specialized GUI items for your usage. Here you see a TreeItem in action.
4. ScrollBar
J2ME Polish uses either a scrollbar (as shown here) or simple scrollindicators to inform the user about possible scrolling.
4. MenuBar
Choose between two menubar options that provide different design possibilities.
