Skylight Conformance to the DOM Specification

Purpose

This document describes the conformance of the Skylight implementation to the Document Object Model (DOM) Level 2 Core Specification. The specification can be found here: http://www.w3.org/TR/2000/REC-DOM-Level-2-Core-20001113.

Conformance Statement

The following is a section by section walkthrough of the first chapter of the specification with comments about the conformance state.

1.1.1

Informative only.

1.1.2

Not considered.

1.1.5

Not implemented directly. It is mapped to the Java type String.

1.1.6

Not implemented.

1.1.7

Not considered.

1.1.8

Not supported.

1.2

See notes about Appendix D and Appendix E for information about the concepts defined in this section.

1.3

The plan is to support only the DOM HTML specification which renders all concepts in this section as unnecessary for an API client.

Non of the concepts defined in this section is implemented in its corresponding couterparts in Appendix D and Appendix E.
